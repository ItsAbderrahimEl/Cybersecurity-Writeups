# 🔐 Penetration Testing Writeups

A personal collection of penetration testing writeups documenting methodology, exploitation techniques, and lessons learned across Hack The Box machines and challenges.

---

## 📁 Repository Structure

```
Penetration-Testing-Writeups/
├── machines/             # Writeups for Hack The Box machines
│   └── machine-name-writeup.md
└── challenges/           # Writeups for Hack The Box challenges
    └── challenge-name-writeup.md
```

---

## 📚 What's Inside Each Writeup

Each writeup documents the full attack path including:

- **Reconnaissance** — enumeration approach and tools used
- **Vulnerability Identification** — findings and their impact
- **Exploitation** — step-by-step exploitation methodology
- **Post-Exploitation** — privilege escalation and lateral movement where applicable
- **Lessons Learned** — key takeaways and techniques worth remembering

---

## 🎯 Categories Covered

**Machines**
- Linux and Windows privilege escalation
- Active Directory attacks
- Service exploitation
- Misconfiguration abuse

**Challenges**
- Web exploitation
- Reverse engineering
- Cryptography
- Binary exploitation
- Forensics
- OSINT
